## 技术决策记录

| 决策 | 选择 | 原因 |
|------|------|------|
| 后端语言 | Go | 高并发、低内存、单二进制部署 |
| Web 框架 | Gin | 轻量、性能好、生态成熟 |
| 数据库 | SQLite | 零运维、单文件、WAL 模式够用 |
| 前端框架 | Tailwind CSS CDN | 无需构建、加载快、设计一致 |
| 图标库 | Lucide Icons | 轻量、风格统一、SVG |
| 模板引擎 | Go html/template | 内置、安全、够用 |
| 密码加密 | bcrypt | 标准、安全 |
