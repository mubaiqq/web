## 页面设计规范

### 色彩体系
- **主色**：Indigo (#6366f1) → Violet (#8b5cf6) 渐变
- **背景**：淡紫渐变 (#eef2ff → #e0e7ff → #f0f9ff → #ede9fe)
- **卡片**：白色半透明 (rgba(255,255,255,0.65)) + backdrop-blur
- **文字**：Gray-900 (主) / Gray-500 (次) / Gray-300 (弱)
- **成功**：Emerald-500
- **错误**：Red-500
- **警告**：Amber-500

### 组件规范
- **圆角**：卡片 1rem / 按钮 0.75rem / 输入框 0.75rem
- **阴影**：`shadow-sm` (卡片) / `shadow-lg` (弹窗) / `shadow-indigo-200` (按钮)
- **间距**：组件间距 1.5rem / 内边距 1.5rem
- **字体**：Inter / SF Pro Display / -apple-system

### 交互规范
- **hover**：卡片上移 2px + 阴影加深
- **focus**：输入框 indigo 边框 + ring 光晕
- **transition**：所有交互 0.15s-0.2s 过渡
- **loading**：骨架屏或 spinner
